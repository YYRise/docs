## 1. 在每行行首或行尾添加 / 删除内容

- 在每行行首添加相同的内容：
`:%s/^/要添加的内容/g`

- 在每行行尾添加相同的内容：
`:%s/$/要添加的内容/g` 

- 利用正则表达式删除代码段每行的行号
`:%s/^\s*[0-9]*\s*//g`

## 2. 删除

- d1G   //删至文档首部
> 1G为跳转至文档首行

- x 删除游标所在的字符
- X 删除游标所在前一个字符
- Delete 同x
- dd 删除整行
- dw 删除一个单词（不适用中文）
- d$或D 删除至行尾
- d^ 删除至行首
- dG 删除到文档结尾处
- 2dd表示一次删除2行

- 删除所有
	- : %d
	> : 切换到命令模式
	- gg dG
	> gg 光标切换到第一行

## 3. 显示当前的文件名、路径
- 文件名： `:f 或CTRL+G`
- 路径: `:!pwd`
